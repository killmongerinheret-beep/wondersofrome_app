# 🚀 Quick Start Guide

## You're Ready to Launch!

All the hard work is done. Here's what to do next:

## Step 1: Test (15 min) ⚡

```batch
TEST_APP.bat
```

Or:
```powershell
cd wondersofrome_app
npm start
```

Then:
- Press `a` for Android
- Press `w` for web browser
- Scan QR with Expo Go app

## Step 2: Build (30 min) 📦

```powershell
cd wondersofrome_app
npx expo run:android --variant release
```

## Step 3: Deploy (1-2 hours) 🌍

1. Go to https://play.google.com/console
2. Pay $25 (one-time)
3. Upload APK
4. Submit!
