================================================================================
TROUBLESHOOTING
================================================================================

❌ "Connection Refused" or "Can't Connect"
   → Check if both devices are on same WiFi
   → Check if Metro bundler is running (look for QR code in terminal)
   → Try the alternative URL above

❌ "Network Error"
   → Turn off mobile data on phone
   → Make sure WiFi is connected
   → Restart Expo Go app

❌ "Invalid URL"
   → Check you typed it exactly: exp://10.106.103.152:8081
   → No spaces, no typos
   → Must start with "exp://"

Tunnel mode is slower but works from anywhere!
