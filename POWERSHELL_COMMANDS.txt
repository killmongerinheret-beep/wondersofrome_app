================================================================================
WHY THE .\ PREFIX?
================================================================================

PowerShell doesn't run files from current directory by default.
The .\ tells PowerShell: "Run this file from HERE"

Examples:
   ❌ Wrong:  START_TUNNEL_MODE.bat
   ✅ Right:  .\START_TUNNEL_MODE.bat

   ❌ Wrong:  CLOUD_BUILD.bat
   ✅ Right:  .\CLOUD_BUILD.bat

================================================================================
ALTERNATIVE: USE CMD INSTEAD
================================================================================

If you prefer, you can use Command Prompt (CMD) instead:

1. Open Command Prompt (not PowerShell)
2. Navigate to folder: cd C:\wondersofrome_app
3. Run without .\:  START_TUNNEL_MODE.bat

In CMD, you DON'T need the .\ prefix!
