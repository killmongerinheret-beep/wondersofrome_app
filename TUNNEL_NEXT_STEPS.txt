================================================================================
IMPORTANT NOTES
================================================================================

⏳ Tunnel is SLOWER than local WiFi:
   - Load time: 10-30 seconds (vs 2-5 seconds local)
   - This is normal for tunnel mode
   - Trade-off for global access

🔒 Tunnel URL is PUBLIC:
   - Anyone with URL can access
   - Only share with trusted testers
   - URL expires after some time

⚡ For Fast Testing:
   - Use local WiFi: exp://10.106.103.152:8081
   - Use tunnel only when needed

================================================================================
TROUBLESHOOTING
================================================================================

If tunnel fails to start:
   - Check internet connection
   - Try again (sometimes takes 2-3 attempts)
   - Check firewall settings

If tunnel is too slow:
   - This is normal
   - Use local WiFi for development
   - Use tunnel only for remote testing

If tunnel URL doesn't work:
   - Make sure you copied the full URL
   - Include "exp://" at the beginning
   - Try scanning QR code instead

================================================================================
WHEN TO USE TUNNEL MODE
================================================================================

✅ Use Tunnel When:
   - Testing from different location
   - Using mobile data
   - Sharing with remote testers
   - Not on same WiFi as computer

❌ Don't Use Tunnel When:
   - Quick local testing
   - Development/debugging
   - Want fast performance
   - On same WiFi as computer

For local testing, use: exp://10.106.103.152:8081
